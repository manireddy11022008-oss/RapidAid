# Technology Stack

## Overview

RapidAid is designed as a modern web application using scalable technologies for frontend, backend, database, AI integration, and cloud services.

---

## Frontend

- React.js
- Tailwind CSS
- HTML5
- CSS3
- JavaScript (ES6+)

### Purpose
Provides a responsive, user-friendly interface for patients, hospitals, and administrators.

---

## Backend

- Node.js
- Express.js

### Purpose
Handles API requests, authentication, emergency routing logic, hospital management, and communication between system components.

---

## Database

- MongoDB

### Purpose
Stores user information, hospital data, ambulance details, emergency requests, medical records, and hazard reports.

---

## AI & Machine Learning

- OpenAI API / Google Gemini API

### Purpose

- Hospital recommendation
- Emergency severity assessment
- Speech-to-text conversion
- AI chatbot assistance
- Route optimization support

---

## Maps & Location Services

- Google Maps API

### Purpose

- Live GPS tracking
- Route visualization
- ETA calculation
- Nearby hospital and ambulance search

---

## Communication Services

- Twilio SMS API

### Purpose

- SMS fallback alerts
- Emergency contact notifications
- Low-network communication support

---

- Git
