# RapidAid Hospital Dashboard UI Design

## Purpose

The Hospital Dashboard enables hospital administrators to manage emergency cases, monitor available resources, update patient status, and coordinate with ambulance teams in real time.

## Page Layout

### Header

- 🚑 RapidAid Logo
- Dashboard
- Emergency Queue
- Resources
- Notifications
- Profile

---

## Hospital Overview

Display:

- Hospital Name
- Hospital ID
- Emergency Status
- Current Capacity
- Last Updated

---

## Emergency Queue

Display:

- Emergency ID
- Patient Name
- Emergency Type
- Severity Level
- ETA
- Assigned Ambulance

Actions:

- Accept Patient
- Prepare Emergency Team
- View Patient Details

---

## Resource Management

Display:

- ICU Beds Available
- General Beds Available
- Emergency Doctors Available
- Nurses Available
- Ventilators Available
- Blood Units Available

Actions:

- Update Availability
- Add Resources
- Mark Resources Unavailable

---

## Patient Information

Display:

- Name
- Age
- Blood Group
- Allergies
- Medical Conditions
- QR Medical Profile

Actions:

- View QR Profile
- Update Treatment Status

---

## Ambulance Tracking

Display:

- Ambulance Number
- Driver Name
- Current Location
- ETA
- Live Route

---

## Notifications

Examples:

- New Emergency Assigned
- Ambulance Arriving in 5 Minutes
- Resource Update Required
- Emergency Completed

---

## Reports

Display:

- Today's Emergency Cases
- Average Treatment Time
- ICU Utilization
- Ambulance Arrivals
- Patient Outcomes
