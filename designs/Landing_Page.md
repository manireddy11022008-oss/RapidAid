# RapidAid Landing Page UI Design

## Purpose

The Landing Page is the first screen users see when they visit RapidAid. It introduces the platform, highlights its key features, and encourages users to register, log in, or learn more about the emergency response system.

---

### How RapidAid Works

1. User presses the Emergency SOS button.
2. Live GPS location is captured.
3. AI analyzes the emergency.
4. Best hospital is recommended.
5. Nearest ambulance is dispatched.
6. Patient tracks ambulance in real time.
7. Hospital prepares before arrival.
8. Patient receives treatment.
